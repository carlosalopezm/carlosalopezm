How do you create a cascading style sheet for your website?

  1. Get an integrated development environment such as Visual Studio Code
  2. Create a document that follows the css naming conventions: use lowercase characters, avoid spaces (use underscores), use alphanumeric characters (0-9,a-z, and A-Z), and end the document with the ".css" extension

How do you link a cascading style sheet to your website?

In the head tags add a link tag with the following properties

  1. Write the opening link tag without closing it: <link
  2. Add a rel equation set to the value of stylesheet: <link rel="stylesheet"
  3. Add a type equation set to the value of text/css: <link rel="stylesheet" type="text/css"
  4. Add an href equation (hyper text reference equation) set to the path assigned to your css file: <link rel="stylesheet" type="text/css" href="./stylesheet.css"
  5. Finaly, do not pair with a closing tag such as </link>. Simply use the following closing tag: />

          </head>


          <body>

Specifying Fonts

Inside your css document you can specify the font family of HTMl elements. To do so simply follow the steps below.

  1. Create a css file and link it to your html file
  2. Write the element identifier (eg p, h1, and h2)
  3. After writing the element identifier add brackets {}
  4. Within the brackeets write font-family followed by a colon: p { font-family: }
  5. After the colon add a space and write the font stack. A font stack is a list of fonts, other than the default sans-serif and serif, that the browser can use in case the first choice isn't installed in the computer.
  6. Finally, add a semicolon

Your css file will look similar to the text below if you wanted to style only the h1 and h2 headers, and the paragraph elements.

h1 {
    font-family: Georgia, "Times New Roman", Times, serif;
}

h2 {
    font-family: Georgia, "Times New Roman", Times, serif;
}

p {
    font-family: Georgia, "Times New Roman", Times, serif;
}




Specifying a Font Size

Inside your css document you can specify the font size of HTMl elements. Right now, I will style the elements of this lesson such that they are 180% of their default size. To do so simply follow the steps below.

  1. Write the element identifier add brackets {}
  2. Within the brackeets write font-size followed by a colon: p { font-family: }
  3. After the colon add a space and write a real number greater than 0 followed by the letters em to specify what percentage of the default size you will use.
  4. Finally, add a semicolon

Your css file will look similar to the text below if you wanted to style only the h1 and h2 headers, and the paragraph elements.

h1 {
    font-family: Georgia, "Times New Roman", Times, serif;
    font-size: 1.2em;
}

h2 {
    font-family: Georgia, "Times New Roman", Times, serif;
    font-size: 0.9em;
}

p {
    font-family: Georgia, "Times New Roman", Times, serif;
    font-size: 1.5em;
}


As shown above the font size of h1 will be 120% of the default, whereas the font-size of h2 and p will be 90% and 150% of their default size, respectively.

Margins


You can choose how much space your text will have at the top, right, left and bottom. To do so follow the steps below.

  1. In your CSS file, write the margin keyword followed by a colon.
  2. The order starts at the top margin and goes clockwise until the left margin: margin: 1em, 2em 1.3em 1.6em
  3. You can also specify the margins separately by writing margin-top, margin-right, margin-bottom, and margin-left: margin-top: 2em

example.html


<!DOCTYPE html>

<html>

          <head>

                <link rel="stylesheet" type="text/css" href="./style.css"/>

          </head>


          <body>

            <p class="bottom-margin"> This paragraph will have a large bottom margin. Hence the space between paragraphs will increase.</p>

            <p class="top-margin"> This paragraph will be further from the upper paragraph than the one below it.</p>

            <p>This paragraph has normal padding.</p>

          </body>

<\html>

style.css

.bottom-margin {
    margin: 1sem 1em 5em 1em;
}

.top-margin {
    margin: 5em;
}

Result:

This paragraph will have a large bottom margin. Hence the space between paragraphs will increase.

This paragraph will be further away from the paragraph above it than it is from the paragraph below it.

Borders


In HTML, you can add a border to any text To do so follow the steps below.

  1. In your CSS file, write the border keyword followed by a colon.
  2. First, you specify the thickness of the border in pixels: border: 5px (no space between the number and px)
  3. Second, you specify the style using either dotted, dashed, solid, double, groove, ridge, inset, and outset: border: 5px solid
  4. Third, you specify the color of the border: border: 5px solid yellow
  5. You can also specify the thickness, style, and color of the top (border-top), right (border-right), bottom (border-bottom), and left (border-left).

example.html


<!DOCTYPE html>

<html>

          <head>

                <link rel="stylesheet" type="text/css" href="./style.css"/>

          </head>


          <body>

            <p class="yellow-border"> This paragraph will have a thick yellow border </p>

            <p class="multi-border"> This paragraph will colorful border of different styles.</p>

          </body>

<\html>

style.css

.yellow-border{
    border: 5px solid yellow;
}

.multi-border {
    border-top: 3px dashed red;     border-right: 6px solid blue;     border-bottom: 8px ridge green;     border-left: 12px dotted orange;
}

Result:

This paragraph will have a thick yellow border

This paragraph will have a colorful border with different styles.


Padding


In HTML, you can add a border with padding to provide a better look to your text.

  1. In your CSS file, write the padding keyword followed by a colon.
  2. First, you specify the top and move clockwise until the left: padding: 2em 1.5em 4em 5em
  3. You can also specify the padding of the top (padding-top), right (padding-right), bottom (padding-bottom), and left (padding-left).

          </body>

<\html>

style.css

.yellow-border-pad{

Result:

This paragraph will have a thick yellow border and nice padding

This paragraph will have a colorful border with different styles and nice padding.
