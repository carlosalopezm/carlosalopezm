Inheritance


Inheritance is an efficiency feature of CSS. This efficiency feature of CSS makes use of the parent and child structure of HTML files. For instance, every element inside the body tags (<body></body>) is a child element of the html body element. Likwise, the HTML head and body are children of the html element (<html></html>).

example.html

<!DOCTYPE html>

<html>

          <head>

                <link rel="stylesheet" type="text/css" href="./style.css"/>

          </head>


          <body>

          <h4> This header is a child element of the body element.</h4>

            <p> This paragraph will be surrounded by a solid 5px blue border and will have a pink background because it is a child elemnt of the html body </p>

          </body>

<\html>

style.css

body{
    background-color: pink;
    border: 5px solid blue;
    padding: 3em 3em 3em 3em;
}

Result:

This header is a child element of the body element.

This paragraph will be surrounded by a solid 5px blue border and will have a pink background because it is a child element of the html body



IDs

In your CSS file, you can specify an ID for elements that require an specific styling. To do so follow the steps below.

  1. In the opening tag of the element taht requires particular styling, write id and equate it to a string with the particular ID.
  2. ID should follow the following conventions: lowercase letters, no spaces (use underscores), use alaphanumeric characters (0-9,a-z, A-Z, _), start with a letter.

example.html

<!DOCTYPE html>

<html>

          <head>

                <link rel="stylesheet" type="text/css" href="./style.css"/>

          </head>


          <body>

          <h4 id="right-blue"> This header is blue and aligned to the right because it has an id of right-blue.</h4>

            <p id="pink-blue"> This paragraph will be surrounded by a solid 5px blue border and will have a pink background because it has an id of pink-blue </p>

          </body>

<\html>

style.css

#pink-blue{
    background-color: pink;
    border: 5px solid blue;
    padding: 2em 2em 2em 2em;
   
}

#right-blue{
    border: 5px solid blue;
    padding-top: 3em;
    padding-right: 3em;
    padding-bottom: 3em;
    padding-left: 3em;
}

Result:

This header is blue and aligned to the right because it has an id of right-blue

This paragraph will be surrounded by a solid 5px blue border and will have a pink background because it has an id of pink-blue

Tables

In your CSS file, you can create tables. Use the 2x2 table below to create new tables with distinct features. Lastly, there will more examples following this one.

Facts about tables

  1. The table is always enclosed between the table tags: <table> </table>
  2. The table rows are enclosed between the tr tags: <tr> </tr>
  3. The table rows are enclosed between the tr tags: <tr> </tr>
  4. The table data is always enclosed between the td tags (<td> </td>) and lie between the table row tags. Hence each td tag between the tr tags represents a column.
  5. The table headers are always enclosed between the th tags: <th> </th>
  6. Each tag can be assigned to a class to style it.
  7. You can provide border to a table in css
  8. If you specify the header and data width, the browser will adjust text acording to the specified table width.
    Otherwise, the browser will simply adjust the header size or data cell to the text's size.: td, th {
    width: 75%
    }
    The example above shows that the data cell and header cell will hav a width equal to 75% of the windows width or 75% of the div's width.
  9. You can align text in the table, a specific row, a specific header, or an specific data cell.
  10. You can make data cells that span more than one row or column by specfying the rowspan or colspan in the opening td tag.

Forms Part 1: Text

In your CSS file, you can create forms that accept text input of a certain character length.

Facts about text forms

  1. Forms are written bewteen the form tags: <form> </form>
  2. You have to add a lable tag: <label>
  3. You have to add an input tag: <input>
  4. The type element inside the input tag has to be set to text: <input type="text" />
  5. The name element inside the input tag has to be set to a reasonable name for your task: <input type="text" name="full-name" />
  6. The size element inside the input tag tells the browser how wide the box should be: <input type="text" name="full-name" size="40" />
  7. The maxlength element inside the input tag tells the browser the maximum number of characters that can be written: <input type="text" name="full-name" size="40" maxlength="50" />

Facts about textarea forms

  2. You have to add a textarea tag: <textarea>
  3. Inside the opening textarea tag specify the name, rows and cols: <textarea name="" rows="" cols="" >
  4. Add a textarea closing tag: </textarea>

Facts about input forms

  1. Forms are written bewteen the form tags: <form> </form>
  2. Between an input tag specify the type to submit, and set the value element equal to some string: <input type="submit" value="Submit form" />
