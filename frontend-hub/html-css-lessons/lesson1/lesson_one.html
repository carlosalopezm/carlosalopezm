Lesson 1 HTML Basics


What is HTML?

The Hyper Text Markup Language is is a document text file that enables the browser you are currently using to assemble a static webpage. One analogy I like to use is one that frames HTML as the manager of a team (of browsers). It is like that one person who cannot speak without uttering imperative sentences such as "put that header there," "place that paragraph there," "make a line break here," etc.


How can I make html files?

  1. Get an integrated development environment such as Visual Studio Code
  2. Create a document that follows the HTML naming conventions: use lowercase characters, avoid spaces (use underscores), use alphanumeric characters (0-9,a-z, and A-Z), and end the document with the ".html" extension

Where are HTML files stored?

All HTML files are stored in a web host's server. However, if you own an important business you may host all your html files in your business' server. With that being said, I must stress that a server is simply a network computer that processes requests for resources such as the HTM file your are currently reading. In other words, you requested this HTML file from the web hosts server and sucessfully acquired it.

Basic HTML Anatomy

Every HTML file follows certain basic conventions:

  1. The first line contains the <!DOCTYPE html> tag
  2. all the code is contaned between the opening and closing html tags: <html> <\html>
  3. There are head tags between the opening html tag and both body tags for metadata or data about data. You would place links to style-sheets, the page title, and other factors that involve metadata within these tags: <head> <\head>
  4. Last but not least, both body tags go between the closing head tag and the closing html tag. These tags contain all the HTML elements that are used to render your webpage: <body> <\body>

Images

You can add images to your website. To do so follow the steps below.

  1. You can display images using the img tag, and setting the src equal to its location in a folder or a url: <img src="./files/myimg.jpeg />
  2. You can also add additional info to your image in case the website fails to load it using alt: <img src"./locaton/my.jpeg" alt="this is my jpeg" />
  3. You can also specify width and height in pixels: <img src"./locaton/my.jpeg" alt="this is my jpeg" width="300" height="450" />
