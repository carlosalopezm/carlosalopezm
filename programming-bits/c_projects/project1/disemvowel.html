Disemvowel

This program is simple but rather useful for a future application that I will make. Essentially, disemvowel removes vowels to make a message seem like complete gibberish. The disemvowel program may resolve trolling, the proliferation of misinformation, or the proliferation of hateful speech. Hence, when I build a more sophisticated system of code that processes language, functions such as disemvowel will be implemented. Enjoy this small snippet of code.




As demonstrated above, the string in main has all vowels removed such that you will probably never understand the message. However, a more interesting application may be one that reintegrates vowels without knowledge of the initial string. Hence, I will try to make another program that integrates vowels.
