Block Chain Data Base

For this project I used C to make a database for tweets. This is rather simple since strings will be the only primitive type used. In case, you were wondering, I am not even using strings but rather arrays of chars. Hence a pointer to an array of chars is used. This project is just a small taste of what can be done with structs in C. I hope you enjoy following the code along with my detailed explanation.
